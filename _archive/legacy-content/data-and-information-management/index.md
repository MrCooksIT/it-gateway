# Data and Information Management

Data and information management covers database design principles, SQL, data integrity, and how organisations collect, store, and use data to support decision-making.

## What you will learn

| Grade | Focus |
|---|---|
| 10 | Data vs information, basic database concepts, simple queries |
| 11 | Relational databases, SQL (SELECT, INSERT, UPDATE, DELETE), forms and reports |
| 12 | Normalisation, database design, data management, big data, data warehousing |

## Topics in this section

- [Database Design and Concepts](./database-concepts) — Normalisation, relationships, keys, ACID properties (Gr 12)
- [Database Management](./database-management) — Data protection, validation, SQL injection, roles (Gr 12)

## CAPS alignment

This topic appears in both papers. Paper 2 tests theoretical knowledge (design, normalisation, integrity). Paper 1 tests practical SQL skills within a programming solution.
