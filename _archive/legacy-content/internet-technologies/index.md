# Internet Technologies

Internet technologies cover the interconnected technologies that enable global networks — including internet services, web technologies, online applications, and internet security.

## What you will learn

| Grade | Focus |
|---|---|
| 10 | Internet services (storage, shopping, social media), basic web concepts |
| 11 | Static vs dynamic websites, compression, internet services technologies |
| 12 | Online applications, data storage (cookies, cloud), web security, streaming |
