# Communication Technologies

Communication technologies cover the network technologies used to transmit digital data — from basic network types and hardware, through to protocols, wireless technologies, and network security.

## What you will learn

| Grade | Focus |
|---|---|
| 10 | Network types (LAN/WAN), network hardware, basic protocols, communication media |
| 11 | Network security, threats and safeguards, wireless technologies |
| 12 | Setting up networks, remote access, advanced security, e-communication security |

## Topics in this section

- [Setting Up Networks](./setting-up-networks) — Essential components, IP addressing, remote access (Gr 12)
- [Electronic Communications Security](./e-communication-security) — Encryption, SSL/TLS, digital certificates (Gr 12)

## CAPS alignment

This topic appears in Paper 2 and is one of the most heavily examined theory areas. Network security concepts overlap with Systems Technologies (computer management) and Social Implications (cybercrime).
