# Communication Technologies — Grade 8 & 9 Revision

## Key concepts to know

- What a **network** is and why we use them
- The difference between **LAN** and **WAN**
- Basic network hardware (router, switch, modem, NIC)
- **Wired vs wireless** networks — advantages and disadvantages
- What an **ISP** is and how we connect to the internet
- Basic network security (passwords, firewalls, antivirus)

## Quick definitions

| Term | Meaning |
|---|---|
| Network | Two or more devices connected to share data and resources |
| LAN | Local Area Network — covers a small area like a school or home |
| WAN | Wide Area Network — covers a large area, e.g. the internet |
| Router | Connects a local network to the internet and routes data |
| Modem | Converts digital data to a signal for transmission (and back) |
| Bandwidth | The maximum amount of data that can be sent in a given time |
| ISP | Internet Service Provider — company that provides internet access |
| Firewall | Security system that monitors and controls network traffic |

## Practice questions

1. Explain the difference between a LAN and a WAN. Give an example of each.
2. A school has 30 computers connected to a central printer. What type of network is this?
3. List **two** advantages and **two** disadvantages of wireless networks compared to wired networks.
4. What is the role of a router in a network?
5. Explain what an ISP does and why you need one to access the internet.
6. Name **two** ways to protect a home wireless network.
7. Give **one** reason why a business might prefer a wired network over a wireless one.

## Typical test format

Your test on this topic will likely include:
- Definitions and abbreviations
- Diagrams (label a network setup)
- Scenario questions (recommend a network type for a given situation)
- Advantages and disadvantages lists
