# Internet Technologies — Grade 8 & 9 Revision

## Key concepts to know

- The difference between the **internet** and the **World Wide Web (WWW)**
- Common internet services (email, cloud storage, online shopping, social media)
- How **URLs** work (protocol, domain name, path)
- What **browsers** and **search engines** are
- Online safety and responsible use
- What **HTML** is at a basic level

## Quick definitions

| Term | Meaning |
|---|---|
| Internet | A global network connecting millions of devices |
| WWW | World Wide Web — a collection of websites accessed via the internet |
| URL | Uniform Resource Locator — the address of a web page |
| Browser | Software used to access and view websites (Chrome, Firefox, Edge) |
| Search engine | Tool that helps you find information on the web (Google, Bing) |
| HTML | HyperText Markup Language — the language used to create web pages |
| Cloud storage | Storing files on remote servers accessed via the internet |
| Download | Receiving data from the internet to your device |
| Upload | Sending data from your device to the internet |

## Practice questions

1. Explain the difference between the internet and the World Wide Web.
2. Give **three** examples of internet services and briefly explain each.
3. What does the URL `https://www.school.co.za/about` tell you? Break it down.
4. Explain the difference between a browser and a search engine.
5. List **two** advantages and **two** risks of using cloud storage.
6. A learner receives an email asking for their password. What should they do and why?
7. What is HTML? Give one example of what an HTML tag does.

## Typical test format

Your test on this topic will likely include:
- Definitions and abbreviations
- URL interpretation questions
- Advantages/disadvantages of internet services
- Online safety scenarios
