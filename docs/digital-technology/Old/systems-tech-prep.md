# Systems Technologies — Grade 8 & 9 Revision

## Key concepts to know

- The difference between **hardware** and **software**
- The **input → processing → output → storage** model
- Types of computers (desktop, laptop, tablet, smartphone, server)
- Types of software (system software vs application software)
- What an **operating system** does
- Common input and output devices and their purposes

## Quick definitions

| Term | Meaning |
|---|---|
| Hardware | Physical components of a computer system |
| Software | Programs and instructions that run on hardware |
| Operating system | System software that manages hardware and runs other programs |
| RAM | Temporary memory used when the computer is on |
| ROM | Permanent memory that holds startup instructions |
| CPU | The processor — carries out instructions |
| Input device | Device used to enter data (keyboard, mouse, scanner) |
| Output device | Device that presents information (monitor, printer, speakers) |

## Practice questions

1. Name **three** input devices and explain what each is used for.
2. Explain the difference between hardware and software. Give one example of each.
3. A learner says "the computer stores all my work in RAM." Is this correct? Explain.
4. What is the role of an operating system? Give one example of an operating system.
5. List **four** types of computers and give a typical use for each.
6. Explain the difference between system software and application software.
7. A school wants to buy computers for a computer lab. List **three** hardware components they should consider and explain why each is important.

## Typical test format

Your test on this topic will likely include:
- Definitions (2–4 marks each)
- Short scenario questions (apply what you know)
- Match the term to the definition
- True/False with explanation
