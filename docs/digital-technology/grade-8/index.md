# Grade 8 Digital Technology

Grade 8 covers two main units. By the end of the year you should understand how computers work and how to be safe and responsible online.

## Units

### Unit 1 — What is Computing?

You will learn where computers came from, how their parts work together, the difference between hardware and software, and where computing is heading in the future.

**Topics:**

## Grade 8 Learning Goals

By the end of Grade 8 you should be able to:

- Trace the history of computers from early mechanical devices to modern AI
- Explain the input → processing → storage → output model with real examples
- Describe the role of the CPU, RAM, ROM, and storage in a computer system
- Distinguish between system software, application software, and utility software
- Identify and describe common input and output devices
- Define what a digital footprint is and explain how to manage yours
- Describe different types of cyberbullying and explain how to respond
- List personal information that should never be shared online
- Create strong passwords and explain why two-factor authentication matters
- Evaluate online information for credibility using the CRAAP test
- Explain copyright, Creative Commons, and what plagiarism means
- Describe the difference between white hat, grey hat, and black hat hackers
