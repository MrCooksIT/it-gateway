# HTML — Structuring Websites

HTML (HyperText Markup Language) is the foundation of every web page on the internet. In this unit you will learn to write HTML from scratch — structuring content, adding links and images, building tables, and applying basic styling.

## The Three Languages of the Web

Web pages are built with three technologies that work together:

| Language | Role | Example |
|----------|------|---------|
| **HTML** | Structure — what content is on the page | Headings, paragraphs, images, links |
| **CSS** | Style — how content looks | Colours, fonts, layout, spacing |
| **JavaScript** | Behaviour — what content does | Click events, animations, form validation |

In this unit we focus on **HTML**. CSS is introduced at the end.

## Learning Goals

By the end of this unit you should be able to:

- Explain what HTML is and how it differs from CSS and JavaScript
- Write a correctly structured HTML page from memory
- Use heading, paragraph, bold, italic, and other text formatting tags
- Create links to other pages and to specific sections on the same page
- Insert images with appropriate alt text
- Build ordered and unordered lists, including nested lists
- Create a table with headers, data, and merged cells
- Apply basic CSS styling using inline styles, internal styles, and external stylesheets

## How to Practise

HTML needs to be practised by writing and running code — not just reading. For each topic:

1. Read the explanations and examples
2. Open a text editor (Notepad, VS Code, or CodeHS)
3. Type the code yourself — don't copy and paste
4. Save the file with a `.html` extension
5. Open it in a web browser to see the result
6. Experiment: change values and see what happens

:::info
You don't need any special software to write HTML. A basic text editor (even Windows Notepad) and any web browser is enough to get started. For a better experience, use VS Code (free) with the "Live Server" extension.
:::
