# Grade 9 Digital Technology

Grade 9 builds on what you learned in Grade 8. You will go deeper into digital citizenship, learn how the web works, and start creating your own web pages with HTML.

## Grade 9 Learning Goals

By the end of Grade 9 you should be able to:

- Explain the POPI Act and what rights it gives South Africans
- Describe offences under the Cybercrimes Act 2020 and their consequences
- Identify advanced social engineering and phishing techniques
- Explain how the internet routes data using TCP/IP and packets
- Describe the role of DNS, HTTP/HTTPS, and web servers
- Break down a URL into its components and explain each part
- Compare different web browsers and explain how they render pages
- Write well-structured HTML pages from scratch
- Use HTML tags for headings, paragraphs, links, images, lists, and tables
- Apply inline and internal CSS to style a web page
