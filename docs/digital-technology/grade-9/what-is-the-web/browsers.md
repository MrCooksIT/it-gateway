# Browsers

## What is a Web Browser?

A **web browser** is a program on your computer that allows you to browse web pages on the internet. When you type a web address into the browser, or click a link, the browser finds the page and displays it on your screen.

:::tip Key Term
**Web Browser:** A software program that runs on your computer and allows you to visit websites on the internet by retrieving and displaying web pages.
:::

It is important to understand what a browser **is** and what it **is not**:

:::info What a browser IS NOT
- A browser is **not** an operating system like Windows or macOS.
- A browser is **not** a website like Google.
:::

:::info What a browser IS
- A browser **is** an application running on your computer that allows you to visit websites on the internet.
:::

## How a Browser Works

### Retrieving the Page

When you type a URL into the browser's address bar and press Enter, the browser retrieves all the files needed to display that web page. This includes:

- **HTML code** — the structure and content of the page
- **CSS code** — the styles that control how it looks
- **Images** — pictures and graphics used on the page
- **Videos** — any video content embedded in the page

### Rendering the Page

Once the browser has downloaded all those files, it does something remarkable: it takes all those resources and **renders** a usable web page from them. Rendering means the browser reads the code and builds the visual page that you see on your screen.

:::info
All of this — retrieving the files and rendering the page — happens in **less than a second**.
:::

---

## Browser Compatibility

Different browsers exist, and there can be some small differences between them. However, for the most part, every browser allows you to browse the internet and view web pages in the same way. This is because all browsers agree to follow the same protocols for how HTML and CSS code should be displayed.

---

## Keeping Your Browser Up to Date

Every web page you view is code running inside your browser. Some web pages may try to attack your computer — installing viruses, tricking you into giving up personal information, and other threats. Keeping your browser up to date is one of the most important things you can do to protect yourself.

An up-to-date browser:

- Has the latest **bug fixes**
- Has the latest **security features**
- Is able to use the most up-to-date versions of **HTML and CSS**
- Likely **runs faster**

:::warning Keep your browser updated
Outdated browsers have known security weaknesses that malicious websites can exploit. Always accept browser updates when they are offered.
:::
