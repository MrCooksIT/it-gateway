# Welcome to Web Design

## Websites Impact the World

Think about all the websites you use in a typical day. Websites are used for:

- **Communication** — messaging, email, social media
- **Education** — tutorials, courses, research
- **Music** — streaming, discovering new artists
- **Art** — galleries, portfolios, creative expression
- **E-commerce** — shopping, selling, banking
- **Access to Knowledge** — encyclopaedias, news, how-to guides

Websites have changed the way the world works. Which websites do you use most commonly? How have they affected your life?

---

## How Are Websites Made?

Websites are made with **code** — specifically, **HTML** and **CSS** code. These two languages work together to create every web page you have ever visited.

---

## Coding is the New Literacy

Think about why we learn to read and write:

> You don't learn to read to become a professional reader. You learn to read because it **enables you to do a lot of other things**.

> You don't learn to write to become a professional writer. You learn to write because it **enables you to do a lot of other things**.

The same is true for coding:

> You don't learn to code to become a professional coder — although you can if you want! You learn to code because it **enables you to do a lot of other things**.

Coding teaches a valuable way of thinking. The skills you develop when you learn to code are useful in any field, not just in technology.

## Web Design vs Web Development

These two roles are related but different:

| | Web Design | Web Development |
|-|-----------|----------------|
| **Focus** | Creating the look and feel of a website | Building the actual site based on a given design |
| **Questions asked** | How should this look? How will users experience this? | How do I write clean, working code to bring this to life? |
| **Main concern** | User experience on the site | Writing clean, readable code |
| **Tools** | Mockups, wireframes, colour palettes | HTML, CSS, JavaScript |

In practice, especially at the beginner level, one person often handles both design and development.
