## How to use this site

| You want to… | Go to… |
|---|---|
| Study for **Paper 2** (theory test) | **Theory** — pick your CAPS topic from the sidebar |
| Prepare for **Paper 1** (practical exam) | **Practical** — work through Delphi or SQL guides |
| Revise quickly before a test | **Quick Study** — cheat sheets, syntax cards, key terms |
| Look up a specific Delphi function | [Delphi Syntax](/quick-study/delphi-syntax) or [String Functions](/quick-study/string-functions) |
| Practice number conversions | [Number Systems](/practical/number-systems/) |

## CAPS Topics at a glance

Each topic is organised **by concept**, not by grade. Grade-level depth is shown inside each page with clear callout boxes — so a Grade 10 learner sees what they need, and a Grade 12 learner sees everything.

| Topic | Paper | Key Grade 12 additions |
|---|---|---|
| Systems Technologies | Paper 2 | Mobile tech, performance factors, cloud |
| Communication Technologies | Paper 2 | Encryption, SSL, network setup |
| Internet Technologies | Paper 2 | Web 3.0, scripting (PHP, JS, XML) |
| Data & Info Management | Paper 2 + Paper 1 (SQL) | Normalisation, joins, data integrity |
| Social Implications | Paper 2 | Environmental impact, distributed computing |
| Solution Development | Paper 1 | OOP (advanced), JOIN queries in Delphi |
