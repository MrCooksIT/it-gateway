# Delphi Programming

Delphi (Object Pascal) is the programming language used in South African IT examinations. It is an event-driven, object-oriented language with a visual IDE called the Delphi RAD Studio (or the free Lazarus equivalent). This section covers everything from placing your first component to writing classes with inheritance.

## CAPS alignment

Delphi programming is the core of **Paper 1** across Grades 10–12. The mark allocation for Delphi typically makes up 50–65% of the total Paper 1 marks.

- **Grade 10** — Variables, basic I/O, selection, loops, simple arrays.
- **Grade 11** — Arrays, string handling, procedures and functions, text files.
- **Grade 12** — OOP, complex algorithms, file handling, database connectivity.
