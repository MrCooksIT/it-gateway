# Practical — Paper 1 Overview

Paper 1 is the **computer-based practical examination** for Information Technology. You will have access to a computer and will write real Delphi code, run SQL queries, and solve problems using the skills you have developed during the year.

This section of IT Gateway covers everything you need for Paper 1, organised by topic: algorithms and problem-solving, Delphi programming, SQL database queries, and number systems.

## What's in this section

| Topic | What you'll learn |
|---|---|
| [Algorithms](./algorithms/) | Designing solutions using flowcharts, pseudocode, sequential, decision and repetition structures |
| [Delphi Programming](./delphi/) | Delphi IDE, components, variables, operators, selection, loops, arrays, strings, files and OOP |
| [SQL](./sql/) | SELECT queries, filtering, sorting, joins, aggregate functions and DML |
| [Number Systems](./number-systems/) | Binary, hexadecimal, conversions, arithmetic and data size calculations |

## How Paper 1 works

- The exam is done on a **computer** supplied by the school or examination centre.
- You will receive a **question paper** and a **data folder** containing Delphi project files and database files.
- Answers are submitted as **saved files** — make sure you save frequently!
- The exam is **3 hours** long.
- Typical mark split: Delphi ~60%, SQL ~20%, Algorithms/Number systems ~20% (varies per year).

## CAPS alignment

Paper 1 assesses the **Solution Development** and **Data & Information Management** (practical SQL) components of CAPS across Grades 10, 11 and 12. The complexity of questions increases each year.
