# Number Systems

Computers store and process everything as numbers — specifically as binary (base 2). Number systems questions test your ability to convert between binary, decimal and hexadecimal, perform simple binary arithmetic, and calculate data sizes. These questions appear in both Paper 1 and Paper 2.

## What's in this section

| Page | What you'll learn |
|---|---|
| [Binary](./binary) | Base-2 number system, converting between binary and decimal, binary arithmetic |
| [Hexadecimal](./hexadecimal) | Base-16 numbers, converting between hex, binary and decimal, why hex is used |
| [Data Sizes](./data-sizes) | Bits, bytes, KB, MB, GB, TB — calculating file sizes and storage requirements |

## Quick conversion reference

| From | To | Method |
|---|---|---|
| Binary → Decimal | Expand place values (powers of 2) | |
| Decimal → Binary | Repeated division by 2, read remainders upward | |
| Binary → Hex | Group bits in fours, convert each group | |
| Hex → Binary | Expand each hex digit to 4 bits | |

## CAPS alignment

Number Systems appears in **Paper 2** (theory questions) across Grades 10–12, and conversion calculations can also appear in Paper 1.

- **Grade 10** — Binary and decimal conversions, understanding bits and bytes.
- **Grade 11** — Hexadecimal conversions, binary addition.
- **Grade 12** — All conversions, data size calculations, two's complement (basic).
