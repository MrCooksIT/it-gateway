# SQL — Structured Query Language

SQL is the language used to interact with relational databases. In the IT examination, you will be given a pre-populated database and asked to write queries that retrieve, filter, sort, group and manipulate data. In Grade 12, you may also be required to connect a Delphi application to a database.

## SQL in the exam

- SQL is usually worth **15–25% of Paper 1 marks**.
- You are given a **database file** (typically Microsoft Access .mdb or .accdb).
- Questions escalate from simple SELECT to multi-table joins and DML.
- Spelling of SQL keywords and field names must be **exact** — a typo means no marks.

## CAPS alignment

SQL is assessed in **Paper 1** across Grades 10–12 under the Data & Information Management practical component.

- **Grade 10** — Basic SELECT with WHERE and ORDER BY.
- **Grade 11** — Aggregate functions, GROUP BY, HAVING, simple joins.
- **Grade 12** — Complex joins, DML (INSERT/UPDATE/DELETE), DDL, Delphi database integration.
