## Internet vs WWW

| | Internet | World Wide Web |
|---|---|---|
| What | Global network of computers | Service running on internet |
| Protocol | TCP/IP | HTTP / HTTPS |
| Content | Email, FTP, VoIP, gaming, web | Web pages |
| Invented | 1969 (ARPANET) | 1991 (Tim Berners-Lee) |

## Web Evolution

| | Web 1.0 | Web 2.0 | Web 3.0 |
|---|---|---|---|
| Era | 1990s–2000s | 2004–now | Emerging |
| Type | Read only | Read-Write | Semantic/AI |
| Content | Owner-created | User-generated | Machine + user |
| Examples | Brochure sites | Facebook, YouTube | AI assistants, IoT |
| Tech | Static HTML | AJAX, APIs, social | Blockchain, AI |

## Static vs Dynamic Websites

| | Static | Dynamic |
|---|---|---|
| Content | Fixed HTML | Generated per request |
| Database | No | Yes |
| Same for all users? | Yes | No — personalised |
| Tech | HTML only | HTML + PHP/Python + DB |
| Example | Simple info site | Online banking, social media |

## Cloud Computing

| Model | What you get | Example |
|---|---|---|
| IaaS | Virtual hardware | AWS, Azure |
| PaaS | Development platform | Google App Engine |
| SaaS | Software via internet | MS 365, Netflix, Gmail |

**Cloud advantages:** anywhere access, auto backup, scalable, no hardware to buy  
**Cloud disadvantages:** internet dependency, privacy concerns, ongoing cost, vendor lock-in

## Online Safety

| Threat | Description |
|---|---|
| Phishing | Fake site/email to steal credentials |
| Identity theft | Using personal info for fraud |
| Cyberbullying | Harassment via technology |
| Malware download | Infected files from unsafe sites |
| Unsecured Wi-Fi | Data intercepted on public networks |

**Protection:** HTTPS, 2FA, VPN on public Wi-Fi, don't overshare, privacy settings on social media

## Cookies

- Small text files stored by browser
- Track visits, preferences, login sessions
- Can be used for targeted advertising
- User can clear cookies or block them
- GDPR/POPIA requires consent notification
