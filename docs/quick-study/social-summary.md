## Copyright & Intellectual Property

| Type | Protects | Duration |
|---|---|---|
| Copyright | Creative works (music, code, writing, art) | Life + 50–70 years |
| Patent | Inventions, processes | 20 years |
| Trademark | Logos, brand names | Renewable indefinitely |

**Copyright gives the owner exclusive right to:** copy, distribute, modify, sell, or licence the work.

## Piracy Types

- End-user piracy (installing on more machines than licensed)
- Counterfeiting (fake copies)
- Internet piracy (illegal downloads/torrents)
- Hard disk loading (pre-installing on PCs for sale)

**Consequences:** fines, criminal charges, civil lawsuit, malware risk

## Cybercrime Types

| Crime | Description |
|---|---|
| Hacking | Unauthorised access to computer/network |
| Virus | Attaches to files; spreads when file shared |
| Worm | Self-replicates across networks; no user action needed |
| Trojan | Disguised as legitimate software; opens backdoor |
| Ransomware | Encrypts files; demands payment |
| Spyware | Secretly monitors user; logs keystrokes |
| Phishing | Fake emails/sites to steal credentials |
| Identity theft | Using another person's personal info for fraud |
| DDoS | Flooding server with traffic to crash it |
| SQL injection | Inserting malicious SQL into web forms |
| Spoofing | Faking sender identity |
| Cyberbullying | Harassing/threatening via technology |

---

## Virus vs Worm vs Trojan

| | Virus | Worm | Trojan |
|---|---|---|---|
| Spreads by | Infected file sharing | Network, no user action | Deceptive download |
| Disguise? | No | No | Yes (looks legitimate) |
| Replicates? | Yes (needs host file) | Yes (self-contained) | No |

## Protection Measures

**Technical:**
- Antivirus / anti-malware
- Firewall (hardware or software)
- Regular software updates/patches
- Encryption + HTTPS
- Two-factor authentication (2FA)
- Strong, unique passwords
- VPN
- Regular backups

**Human:**
- Don't click suspicious links
- Verify sender email addresses
- Never give OTPs/passwords by phone
- Check URL before entering credentials (padlock, https)
- Be cautious about what you share on social media

---

## Phishing Signs

- Urgent language ("Act now or your account will be closed")
- Sender email doesn't match the claimed organisation
- Hover over link — URL doesn't match what's shown
- Grammar/spelling errors
- Requests for password, OTP, or card details

## Digital Divide

The gap between those with and without access to ICT:

**Causes:** cost of devices/connectivity, lack of infrastructure, digital literacy, geography (rural vs urban)

**Effects:** inequality in education, employment, healthcare, government services

## Environmental Impact of ICT

| Issue | Detail |
|---|---|

| Energy consumption | Data centres use enormous electricity |
