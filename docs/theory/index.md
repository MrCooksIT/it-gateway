# Theory — Paper 2 Overview

This section covers the six CAPS topics assessed in the **Paper 2 (Theory)** examination for Information Technology Grades 10–12. Each topic builds year-on-year, so it is important to master the foundational concepts before moving to more advanced content.

## What's in this section

| Topic | What you'll learn |
|---|---|
| [Systems Technologies](./systems/) | Hardware components, software, operating systems, performance, mobile tech, cloud and virtualisation |
| [Communication Technologies](./networks/) | Network types, hardware, topologies, protocols, wireless networking and e-communication |
| [Internet Technologies](./internet/) | How the internet and WWW work, web design, HTML, scripting and online services |
| [Data & Information Management](./databases/) | Data concepts, database design, normalisation, integrity and SQL theory |
| [Social Implications](./social/) | Copyright, cybercrime, online safety, privacy, the digital divide and environmental impact |
| [Solution Development (Theory)](./programming/) | Programming concepts, algorithms, OOP principles and software engineering theory |

## CAPS alignment

All six topics are examined in **Paper 2** across Grades 10, 11 and 12. The depth and complexity increase each year:

- **Grade 10** — Introduces foundational concepts and terminology.
- **Grade 11** — Extends understanding; students begin applying concepts in context.
- **Grade 12** — Requires synthesis, evaluation and application of all prior learning.

Paper 2 is a **written examination** (no computer required). Expect definitions, short-answer questions, scenario analysis and diagram-based questions.
