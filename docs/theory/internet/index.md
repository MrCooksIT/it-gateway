# Internet Technologies

Internet Technologies explores how the internet and World Wide Web function, how websites are built and designed, and the growing range of online services that individuals and organisations rely on. It also touches on the theory behind web development — HTML, CSS and scripting — even though the practical assessment of coding happens in Paper 1.

## What's in this section

| Page | What you'll learn |
|---|---|
| [Internet & WWW](./internet-www) | Difference between the internet and the WWW, URLs, IP addresses, DNS |
| [Web Evolution](./web-evolution) | Web 1.0, Web 2.0, Web 3.0 — how the web has changed and where it is heading |
| [Internet Services](./internet-services) | Browsing, email, FTP, streaming, cloud storage, VoIP and more |
| [Multimedia](./multimedia) | Text, images, audio, video and animation — formats, resolution, file sizes |
| [Website Design](./website-design) | Design principles, usability, accessibility, responsive design |
| [HTML Basics](./html-basics) | Tags, structure, headings, lists, links, images — the building blocks of a web page |
| [Web Scripting](./web-scripting) | Introduction to client-side scripting, JavaScript concepts, form validation |
| [Online Services](./online-services) | E-commerce, e-banking, e-government, online education and social media |

## CAPS alignment

Internet Technologies is assessed in **Paper 2** across Grades 10–12.

- **Grade 10** — Internet vs WWW, browsers, basic HTML structure, online services overview.
- **Grade 11** — Web design principles, multimedia concepts, e-commerce and online banking.
- **Grade 12** — Web evolution, scripting concepts, advanced online services, legal and ethical aspects.
