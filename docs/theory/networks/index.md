# Communication Technologies

Communication Technologies covers how computers and devices are connected together to share data and resources. From a basic home Wi-Fi network to the global internet, this topic explores the hardware, software, standards and security measures that make modern networking possible.

| [Network Hardware](./network-hardware) | NICs, hubs, switches, routers, access points, modems and cabling |
| [Topologies](./topologies) | Bus, star, ring, mesh and tree topologies — diagrams, pros and cons |
| [Protocols](./protocols) | TCP/IP, HTTP/S, FTP, SMTP, POP3, IMAP, DNS — what they do and why they matter |
| [Wireless Networking](./wireless) | Wi-Fi standards (802.11), Bluetooth, infrared, cellular data (3G/4G/5G) |

| [E-Communication](./e-communication) | Email, instant messaging, VoIP, video conferencing, collaborative tools |

## CAPS alignment

Communication Technologies is examined in **Paper 2** across all three grades.

- **Grade 10** — Network types, basic hardware, topologies and the concept of protocols.
- **Grade 11** — Protocols in depth, wireless networking, client-server architecture.
- **Grade 12** — Network security, advanced protocols, VPNs and cloud-based communication.
