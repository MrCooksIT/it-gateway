# Cybercrime

As more of our lives move online — banking, shopping, communication, health records — the potential for harm grows with it. Cybercrime costs the global economy trillions of rands annually. Understanding the threats, how they work, and how to protect against them is core Paper 2 content.

> [!NOTE] Grade 11+
> Cybercrime types, prevention, and legislation are extended in Grade 11. Network security (encryption, SSL) is a Grade 12 focus.

---

## What is Cybercrime?

**Cybercrime** is any criminal activity that involves a computer or network — either as the tool used to commit the crime, or as the target of the crime.

---

## Types of Cybercrime

### Hacking

**Hacking** is gaining unauthorised access to a computer system or network.

| Type | Description |
|---|---|
| **Black hat hacker** | Malicious intent — steal data, cause damage |
| **White hat hacker** | Ethical hacker — paid to find vulnerabilities (penetration testing) |
| **Grey hat hacker** | Breaks in without permission, but typically reports findings |

**Methods hackers use:**
- Exploiting software vulnerabilities (bugs/flaws in code)
- Social engineering (tricking people into revealing credentials)
- Brute force attacks (trying many passwords until one works)
- Man-in-the-middle attacks (intercepting communications)

---

### Malware

**Malware** (malicious software) is any program designed to damage, disrupt, or gain unauthorised access to a system.

| Type | Behaviour | How it spreads |
|---|---|---|
| **Virus** | Attaches to a legitimate file; activates when file is opened | File sharing, infected downloads |
| **Worm** | Self-replicates across networks without user action | Network vulnerabilities |
| **Trojan horse** | Disguised as legitimate software; opens a backdoor | Malicious downloads, fake software |
| **Ransomware** | Encrypts files; demands payment for decryption key | Phishing emails, infected sites |
| **Spyware** | Secretly monitors user activity, logs keystrokes | Bundled with free software |
| **Adware** | Displays unwanted advertisements | Bundled software, malicious websites |
| **Rootkit** | Hides deep in OS, giving persistent hidden access | Advanced malware |
| **Keylogger** | Records all keystrokes to capture passwords | Installed by Trojans or physical access |
| **Botnet** | Network of infected "zombie" computers controlled remotely | Worms, Trojans |

---

### Phishing

**Phishing** is a social engineering attack that tricks users into revealing sensitive information (passwords, credit card numbers, ID numbers) by pretending to be a trusted entity.

**How phishing works:**
1. Victim receives an email appearing to be from their bank/Netflix/SARS
2. Email contains an urgent message and a link
3. Link leads to a fake website that looks identical to the real one
4. Victim enters their credentials — now the attacker has them

**Variations:**
- **Spear phishing**: Targeted phishing using personal information to seem credible
- **Vishing**: Voice phishing — phone calls pretending to be from banks/SARS
- **Smishing**: SMS phishing — fraudulent text messages
- **Whaling**: Phishing targeting senior executives/high-value targets

---

### Identity Theft

**Identity theft** is obtaining and using someone else's personal information — ID number, banking details, passwords — to commit fraud or other crimes.

**How information is obtained:**
- Phishing attacks
- Data breaches
- Malware (keyloggers, spyware)
- Shoulder surfing
- Dumpster diving (finding discarded statements/documents)
- Social media oversharing

**Consequences for victims:**
- Fraudulent bank transactions
- New accounts or loans opened in their name
- Criminal records attached to their identity
- Years to resolve and restore credit

---

### Other Cybercrime Types

| Crime | Description |
|---|---|
| **Cyberbullying** | Using technology to harass, threaten, or humiliate individuals |
| **Online fraud** | Fake online stores, lottery scams, advance fee fraud (419 scams) |
| **DDoS attack** | Distributed Denial of Service — flooding a server with requests to crash it |
| **SQL injection** | Inserting malicious SQL code into a web form to access/manipulate a database |
| **Spoofing** | Faking the sender identity of an email/message or IP address |
| **Piracy** | Distributing copyrighted software, music, film without authorisation |
| **Cyber espionage** | Stealing confidential government or corporate data |
| **Sexting / image-based abuse** | Distributing intimate images without consent |

## Protecting Against Cybercrime

### Technical Measures

| Measure | What it does |
|---|---|
| **Antivirus / anti-malware** | Detects and removes malicious software |
| **Firewall** | Monitors and filters incoming/outgoing network traffic |
| **Software updates / patches** | Fixes known security vulnerabilities |
| **Encryption** | Scrambles data so only authorised parties can read it |
| **HTTPS / SSL** | Encrypts web traffic between browser and server |
| **Two-factor authentication (2FA)** | Requires second verification (OTP, app) beyond password |
| **Strong passwords** | Long, complex, unique — not reused across sites |
| **Password manager** | Generates and stores unique complex passwords |
| **VPN** | Encrypts all internet traffic; hides IP address |
| **Backup** | Regular copies of data so ransomware attacks can be recovered from |

### Human Measures (Education)

| Practice | Why |
|---|---|
| Never click suspicious links in emails | Prevents phishing |
| Verify the sender's actual email address | Spoofed display names look real |
| Never give passwords or OTPs over phone | Banks never ask for this |
| Use 2FA on important accounts | Even compromised passwords need a second factor |
| Keep software updated | Patches close exploited vulnerabilities |
| Be cautious of what you share on social media | Reduces spear phishing effectiveness |
| Use HTTPS sites for any sensitive transaction | Prevents man-in-the-middle interception |

## South African Legislation

| Act | Relevance |
|---|---|
| **Electronic Communications and Transactions Act (ECT Act, 2002)** | Criminalises hacking, data interception, computer misuse |
| **Cybercrimes Act (19 of 2020)** | More comprehensive; covers ransomware, unlawful access, fake information |
| **POPIA (Protection of Personal Information Act, 2013)** | Organisations must protect personal data; failure = fines up to R10 million |

## Exam Focus

> [!IMPORTANT] High-Frequency Questions
>
> 1. **"Explain what phishing is and describe how to identify a phishing email"** — fake trusted sender, urgent message, suspicious link; identify by: hover over link (URL mismatch), sender email doesn't match organisation, grammar errors, unsolicited request
>
> 2. **"Name and describe THREE types of malware"** — know virus, worm, Trojan, ransomware, spyware — always describe what each does specifically
>
> 3. **"Give FOUR measures to protect a computer against cybercrime"** — can mix technical (antivirus, firewall, 2FA, updates) with human (strong passwords, don't click links)
>
> 4. **"What is the difference between a virus and a worm?"** — virus: attaches to files, needs user action to spread; worm: self-replicates, spreads automatically through networks
>
> 5. **"Why is two-factor authentication more secure than a password alone?"** — even if password is stolen/guessed, attacker still needs second factor (OTP, physical token) which they don't have
