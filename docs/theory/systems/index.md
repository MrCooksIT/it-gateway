# Systems Technologies

Systems Technologies is the foundation of the IT theory curriculum. It covers what computers are made of (hardware), how they are controlled (software and operating systems), how performance is measured and improved, and how modern trends like cloud computing and virtualisation are changing the way we use technology.

## CAPS alignment

Systems Technologies appears in **Paper 2** and is examined across **Grades 10, 11 and 12**.

- **Grade 10** — Hardware components, peripherals, types of computers, basic software concepts.
- **Grade 11** — Operating system management, performance factors, mobile technology.
- **Grade 12** — Cloud computing, virtualisation, advanced OS concepts, data representation.
