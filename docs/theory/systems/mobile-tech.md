# Mobile Technology

Mobile technology has transformed how we communicate, work, learn, and entertain ourselves. A device in your pocket has more computing power than the computers that landed humans on the moon — and it is always connected.

> [!NOTE] Grade 12
> Mobile technology is primarily a Grade 12 topic, though mobile devices are introduced in Grade 10 as a type of computer. Grade 12 focus: advantages/limitations of mobile, 4G/5G, mobile OS, mobile security.

---

## What is Mobile Technology?

**Mobile technology** refers to portable computing devices and the wireless networks that connect them, enabling communication and computing from any location.

## Mobile Operating Systems

### Android (Google)
- Open source — manufacturers can customise it
- Used by Samsung, Huawei, Xiaomi, and many others
- Largest market share globally
- Apps from Google Play Store
- More customisable; slightly more malware risk due to sideloading

### iOS (Apple)
- Closed source — only on Apple iPhones and iPads
- Tightly controlled ecosystem
- Apps only from Apple App Store (strict approval process)
- Generally more secure and consistent updates
- Less customisable

### Comparison

| Feature | Android | iOS |
|---|---|---|
| Manufacturer | Multiple (Samsung, Huawei, etc.) | Apple only |
| Source | Open source | Closed source |
| App store | Google Play | Apple App Store |
| Customisation | High | Limited |
| Market share | ~72% global | ~27% global |
| Updates | Varies by manufacturer | Guaranteed for years |
| Price range | Budget to premium | Premium only |

## Mobile Network Generations

| Generation | Speed | Latency | Key Feature |
|---|---|---|---|
| **2G (GSM)** | ~0.1 Mbps | High | Voice calls + SMS |
| **3G** | ~2–42 Mbps | ~100ms | Mobile internet |
| **4G LTE** | ~10–150 Mbps | ~30–50ms | HD video, mobile apps |
| **5G** | ~100 Mbps–10 Gbps | <10ms | IoT, autonomous vehicles, AR/VR |

### 4G vs 5G

| Feature | 4G LTE | 5G |
|---|---|---|
| Max speed | ~150 Mbps | ~10 Gbps |
| Latency | 30–50 ms | <1 ms |
| Capacity | Moderate density | Very high device density |
| Infrastructure | Existing towers | New mmWave towers + existing |
| Coverage | Widespread | Still expanding (2024) |
| Use cases | Streaming, browsing | Autonomous vehicles, smart cities, AR |

## Advantages of Mobile Computing

| Advantage | Explanation |
|---|---|
| **Portability** | Work, communicate, access data from anywhere |
| **Always connected** | 4G/5G/Wi-Fi — constant internet access |
| **GPS and location** | Navigation, location-based services |
| **Integrated camera** | Photography, video, document scanning |
| **Health monitoring** | Steps, heart rate, sleep tracking (wearables) |
| **Cloud sync** | Data backed up and accessible across all devices |
| **App ecosystem** | Millions of apps for every purpose |
| **Contactless payment** | Apple Pay, Google Pay — pay without a card |

---

## Disadvantages and Limitations of Mobile Computing

| Limitation | Explanation |
|---|---|
| **Battery life** | Powerful features drain battery; constant charging needed |
| **Small screen** | Limits usability for complex tasks |
| **Input method** | Touchscreen typing is slower than physical keyboard |
| **Processing power** | Less powerful than desktop/laptop for same price |
| **Storage** | Internal storage often limited (mitigated by cloud) |
| **Security risks** | Physical theft; malware apps; public Wi-Fi interception |
| **Connectivity dependence** | Many features unusable without internet |
| **Health concerns** | Eye strain, posture, radiation (minimal but debated) |
| **Distraction** | Notifications reduce focus and productivity |

## Mobile Security Threats

| Threat | Description |
|---|---|
| **Device theft** | Physical loss of device gives attacker access to data |
| **Malicious apps** | Apps that contain malware (especially from unofficial sources) |
| **Public Wi-Fi** | Unencrypted connections can be intercepted |
| **Phishing via SMS** | "Smishing" — fake text messages with malicious links |
| **Spyware** | Apps that secretly track location, messages, calls |
| **Ransomware** | Increasingly targeting mobile devices |

### Mobile Security Best Practices
- Use a strong PIN, password, or biometric lock
- Only install apps from official stores (Play Store / App Store)
- Keep OS and apps updated
- Use VPN on public Wi-Fi
- Enable remote wipe (Find My iPhone / Find My Device)
- Be cautious of app permissions — does a calculator need access to contacts?

## Internet of Things (IoT)

**IoT** is the network of physical objects embedded with sensors, software, and connectivity that allows them to collect and exchange data.

**Examples:**
- Smart home devices (lights, thermostat, locks)
- Smart city traffic management
- Wearable health monitors
- Industrial sensors in factories
- Smart agriculture (soil moisture sensors)

IoT devices connect via Wi-Fi, Bluetooth, Zigbee, or 5G. 5G's low latency and high device density make it ideal for IoT.

## Exam Focus

> [!IMPORTANT] High-Frequency Questions
>
> 1. **"Give THREE advantages of mobile computing"** — portability, always connected, GPS, camera, cloud sync — always explain, don't just list
>
> 2. **"Give THREE limitations/disadvantages of mobile computing"** — battery life, small screen, security risks, processing power, connectivity dependence
>
> 3. **"Compare Android and iOS"** — open source vs closed; multiple manufacturers vs Apple only; customisation; app store control
>
> 4. **"Compare 4G and 5G"** — speed (5G much faster), latency (5G <10ms vs 4G 30–50ms), coverage (4G wider), use cases (5G enables IoT, autonomous vehicles)
>
> 5. **"What is IoT? Give TWO examples."** — network of physical objects with sensors and connectivity; examples: smart home devices, wearable health monitors, industrial sensors
