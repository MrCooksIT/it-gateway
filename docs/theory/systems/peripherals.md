# Peripherals — Input, Output, and Storage Devices

The CPU and memory are the brain and short-term memory of a computer. Peripherals are everything else — the devices that let you put information in, get results out, and store data permanently.

> [!NOTE] Grade 10+
> Peripheral devices are introduced in Grade 10 as part of the hardware unit. The classification and examples are core Paper 2 content.

---

## What is a Peripheral?

A **peripheral** is any external device connected to a computer that provides input, output, communication, or storage. The CPU and RAM are not peripherals — everything else typically is.

**Categories:**
- **Input devices** — send data INTO the computer
- **Output devices** — display/produce data FROM the computer
- **Storage devices** — store data permanently (can be both input and output)
- **Communication devices** — connect the computer to networks or other devices

---

## Input Devices

An **input device** converts real-world data into a digital signal the CPU can process.

| Device | Type | How it works | Use |
|---|---|---|---|
| **Keyboard** | Alphanumeric | Each key press sends a digital code | Typing text and commands |
| **Mouse** | Pointing | Tracks movement; buttons send click signals | Navigate GUI, select items |
| **Trackpad / touchpad** | Pointing | Capacitive surface detects finger movement | Laptop navigation |
| **Touchscreen** | Pointing + gesture | Capacitive or resistive surface detects touch coordinates | Smartphones, tablets, ATMs |
| **Barcode scanner** | Optical | Laser reads light/dark patterns of a barcode | Retail, inventory, library |
| **QR code reader** | Optical | Camera decodes 2D matrix patterns | Menus, payments, links |
| **Scanner (flatbed)** | Optical | CCD captures reflected light from a document | Digitise documents and images |
| **Webcam / camera** | Optical | Captures video/image frames | Video calls, photos, surveillance |
| **Microphone** | Audio | Converts sound waves to electrical signal | Voice input, recording |
| **Digital camera** | Optical | CCD/CMOS sensor captures image | Photography |
| **Joystick / gamepad** | Pointing | Analogue movement + buttons | Gaming, flight simulation |
| **Stylus / graphics tablet** | Pointing | Pressure-sensitive drawing surface | Digital art, CAD |
| **Biometric reader** | Biometric | Fingerprint, iris, face recognition | Security authentication |
| **OCR scanner** | Optical | Recognises printed text and converts to editable text | Document digitisation |
| **MICR reader** | Magnetic | Reads magnetic ink characters (bottom of cheques) | Bank cheque processing |
| **Sensors** | Environmental | Temperature, light, motion — converts physical to digital | IoT, embedded systems |

---

## Output Devices

An **output device** receives processed data from the CPU and presents it to the user.

| Device | Type | Detail | Use |
|---|---|---|---|
| **Monitor (LCD/LED)** | Visual | Pixels lit by backlight; LED is energy-efficient | Primary visual output |
| **OLED monitor** | Visual | Each pixel generates its own light; deeper blacks | High-end displays |
| **Projector** | Visual | Projects enlarged image onto a surface | Presentations, cinema |
| **Printer (inkjet)** | Printed | Sprays tiny ink droplets onto paper | Home/office documents |
| **Printer (laser)** | Printed | Uses static electricity + toner powder | High-volume office printing |
| **3D printer** | Printed | Deposits material layer by layer | Prototyping, manufacturing |
| **Speakers** | Audio | Convert digital audio signal to sound waves | Music, alerts, voice |
| **Headphones** | Audio | Personal audio output | Private listening |
| **Plotter** | Visual | Draws precise lines — pen moves on paper | Engineering drawings, CAD |
| **Braille display** | Tactile | Raises/lowers pins to form Braille characters | Accessibility for visually impaired |

---

## Inkjet vs Laser Printer

| Feature | Inkjet | Laser |
|---|---|---|
| Technology | Liquid ink sprayed onto paper | Powdered toner fused by heat |
| Cost of printer | Lower | Higher |
| Cost per page | Higher (ink cartridges expensive) | Lower (toner lasts longer) |
| Speed | Slower | Faster |
| Quality | Excellent for photos, colour | Excellent for sharp text |
| Best for | Home use, photos, occasional colour | High-volume office printing |

## Storage Devices

Storage devices hold data **permanently** (non-volatile) — data persists when power is off.

| Device | Technology | Speed | Capacity | Use |
|---|---|---|---|---|
| **HDD (Hard Disk Drive)** | Magnetic spinning platters | ~100–160 MB/s | Up to 20 TB | Cheap bulk storage |
| **SSD (Solid State Drive)** | Flash memory chips | 500 MB/s–7 GB/s | Up to 4 TB common | Fast OS/app storage |
| **USB Flash Drive** | Flash memory | Moderate | 4 GB–2 TB | Portable file transfer |
| **SD Card / microSD** | Flash memory | Varies | Up to 1 TB | Cameras, phones, tablets |
| **Optical disc (DVD)** | Laser reads/writes | Slow | 4.7 GB (DVD) | Media distribution |
| **Optical disc (Blu-ray)** | Laser reads/writes | Slow | 25–100 GB | HD video, archiving |
| **Magnetic tape** | Sequential magnetic | Very slow access | Hundreds of TB | Archival backup |
| **Cloud storage** | Remote servers | Depends on internet | Virtually unlimited | Backup, sync, sharing |

---

## Communication Devices

These connect computers to networks or other devices.

| Device | Function |
|---|---|
| **Network Interface Card (NIC)** | Connects computer to wired (Ethernet) network |
| **Wireless NIC (Wi-Fi card)** | Connects to wireless networks |
| **Modem** | Converts digital data to analogue for phone lines (and back) |
| **Router** | Connects networks; directs data packets between networks; provides DHCP |
| **Switch** | Connects devices on the SAME network; faster than hub |
| **Hub** | Connects devices on same network; broadcasts to all (less efficient than switch) |
| **Wireless Access Point (WAP)** | Extends wireless network coverage |
| **Bluetooth adapter** | Short-range wireless connection for peripherals |
| **4G/5G modem/dongle** | Mobile internet connection via cellular network |

## Ports and Connectors

| Port | Use |
|---|---|
| **USB-A** | Standard USB; most peripherals |
| **USB-C** | Modern connector; supports data + video + power; reversible |
| **HDMI** | High-definition video and audio output to monitor/TV |
| **DisplayPort** | Video output (often used for high-refresh monitors) |
| **Ethernet (RJ-45)** | Wired network connection |
| **3.5mm audio jack** | Headphones/speakers/microphone |
| **SD card slot** | Insert SD cards directly |
| **Thunderbolt** | Very high speed — data + video + charging in one port |

---

## Plug and Play (PnP)

**Plug and Play** is the ability of an operating system to automatically detect and configure a newly connected device — no manual driver installation by the user.

1. User plugs in device
2. OS detects new hardware
3. OS searches for and installs the correct driver
4. Device is ready to use

Most modern devices (USB peripherals, monitors, printers) are Plug and Play.

## Key Terms

| Term | Definition |
|---|---|
| **Peripheral** | Any device connected to a computer (not the CPU or RAM) |
| **Input device** | Device that sends data into the computer |
| **Output device** | Device that presents processed data from the computer |
| **Storage device** | Device that permanently holds data |
| **HDD** | Hard Disk Drive — magnetic storage with spinning platters |
| **SSD** | Solid State Drive — flash memory, no moving parts |
| **OCR** | Optical Character Recognition — scanning printed text to editable text |
| **MICR** | Magnetic Ink Character Recognition — reads magnetic ink on cheques |
| **Modem** | Converts digital data to analogue signals for phone line transmission |
| **NIC** | Network Interface Card — hardware enabling network connection |
| **WAP** | Wireless Access Point — extends Wi-Fi coverage |
| **Plug and Play** | OS automatically detects and configures new hardware |

## Exam Focus

> [!IMPORTANT] High-Frequency Questions
>
> 1. **"Classify each device as input/output/storage and give a reason"** — the reason must link to the direction of data flow, not just the device name
>
> 2. **"Give ONE example of each: input, output, storage device"** — must give correct examples; printer is output, hard drive is storage, keyboard is input
>
> 3. **"Compare inkjet and laser printers"** — minimum two criteria: cost per page (laser cheaper), speed (laser faster), quality (laser text, inkjet photo)
>
> 4. **"What is the difference between a hub and a switch?"** — hub broadcasts to all devices; switch sends only to the intended device (more efficient, faster)
>
> 5. **"Give an advantage of an SSD over an HDD"** — no moving parts, faster, quieter, more shock-resistant, uses less power
