# Types of Computers

Not all computers look like the desktop in a lab. A microwave, a smartwatch, and a supercomputer are all computers — they all receive input, process it, and produce output. What differs is their size, power, purpose, and portability.

> [!NOTE] Grade 10+
> Types of computers and their characteristics are introduced in Grade 10. Mobile technology and embedded systems extend into Grade 12.

---

## Classification by Size and Power

### Microcomputers (Personal Computers)

The most common type — used by individuals. All the devices below are microcomputers.

---

### Desktop Computer

| Aspect | Detail |
|---|---|
| **Location** | Fixed on a desk — not portable |
| **Components** | Separate: tower/case, monitor, keyboard, mouse |
| **Power** | Plugged into mains power |
| **Performance** | High — larger components, better cooling, upgradeable |
| **Use** | Offices, labs, gaming, graphic design |
| **Advantages** | Upgradeable, better cooling, larger screen, cheaper per spec |
| **Disadvantages** | Not portable, requires desk space |

---

### Laptop (Notebook)

| Aspect | Detail |
|---|---|
| **Components** | Integrated: screen, keyboard, touchpad in one unit |
| **Power** | Battery + power adapter |
| **Performance** | Good — slightly lower than desktop for same price |
| **Use** | Students, professionals, home users who need mobility |
| **Advantages** | Portable, built-in battery, all-in-one |
| **Disadvantages** | Less upgradeable, smaller screen, can overheat, battery life limited |

---

### Tablet

| Aspect | Detail |
|---|---|
| **Interface** | Touchscreen — no physical keyboard (can attach external) |
| **OS** | Android, iPadOS, Windows |
| **Power** | Battery |
| **Use** | Reading, media consumption, light productivity, education |
| **Advantages** | Very portable, lightweight, long battery life, intuitive touch |
| **Disadvantages** | Less powerful than laptop, limited software, typing is slower |

---

### Smartphone

| Aspect | Detail |
|---|---|
| **Interface** | Touchscreen, voice |
| **OS** | Android, iOS |
| **Connectivity** | 4G/5G, WiFi, Bluetooth, GPS |
| **Use** | Communication, social media, navigation, mobile banking, photography |
| **Advantages** | Always connected, GPS, camera, extremely portable |
| **Disadvantages** | Small screen, limited processing power vs laptop, security risks |

---

### Smartwatch / Wearable

| Aspect | Detail |
|---|---|
| **Interface** | Small touchscreen, voice |
| **Use** | Health monitoring (HR, steps), notifications, contactless payment |
| **Limitations** | Small screen, limited input, depends on paired phone |
| **Examples** | Apple Watch, Samsung Galaxy Watch, Fitbit |

## Server

A **server** is a powerful computer that provides services to other computers (clients) on a network.

| Type | Purpose |
|---|---|
| **File server** | Stores and shares files across a network |
| **Web server** | Hosts websites, serves web pages to browsers |
| **Mail server** | Manages email (sending, receiving, storing) |
| **Database server** | Runs a database and responds to queries |
| **Print server** | Manages shared printers on a network |
| **Application server** | Runs applications and delivers them to clients |

**Server characteristics:**
- Runs 24/7 (uptime is critical)
- No monitor or keyboard usually attached (managed remotely via CLI)
- Multiple CPU cores and large RAM
- Redundant storage (RAID) to prevent data loss
- Runs a **Network Operating System (NOS)** like Windows Server or Linux

---

## Mainframe

| Aspect | Detail |
|---|---|
| **Size** | Large room-filling systems |
| **Users** | Hundreds to thousands simultaneously |
| **Use** | Banking transactions, airline reservations, government databases |
| **Performance** | Exceptionally reliable, processes millions of transactions per second |
| **Examples** | IBM Z Series |

---

## Supercomputer

| Aspect | Detail |
|---|---|
| **Speed** | Fastest computers in existence — measured in FLOPS (floating point operations per second) |
| **Use** | Weather forecasting, nuclear simulation, climate modelling, drug research |
| **Cost** | Millions to billions of rands |
| **Power** | Consumes enormous electricity; requires dedicated cooling |
| **Examples** | Frontier (USA), Fugaku (Japan) |

## Embedded Systems

An **embedded system** is a computer built into a device for a specific, dedicated purpose. The user doesn't see it as a "computer."

| Device | Embedded Computer Function |
|---|---|
| Smart TV | Process remote commands, stream video, display output |
| Microwave oven | Timer management, power level control |
| Modern car | Engine management, ABS, GPS navigation, entertainment |
| ATM machine | Process banking transactions, control card reader |
| Medical pacemaker | Monitor heart rate, deliver shocks when needed |
| Traffic light | Control timing sequence, respond to sensors |
| Industrial robot | Precise movement control, sensor response |

**Characteristics:**
- Dedicated single purpose — can't be reprogrammed by user
- Real-time OS (must respond instantly)
- Small, energy-efficient, often no display
- Highly reliable (failure can be dangerous)

## Mobile Technology

> [!IMPORTANT] Grade 12 — Mastery
> Mobile technology and its advantages/limitations are a Grade 12 focus topic.

### Advantages of Mobile Devices
- Portability — take your work/data anywhere
- Always connected via 4G/5G/WiFi
- GPS and location services
- Integrated camera, microphone, sensors
- Long battery life vs laptop
- Cloud sync keeps data updated across devices

### Limitations of Mobile Devices
- **Processing power** — less than desktop/laptop
- **Storage** — limited internal storage (mitigated by cloud)
- **Screen size** — small display limits usability
- **Input method** — touchscreen typing slower than physical keyboard
- **Battery life** — still limited, needs regular charging
- **Security** — more vulnerable to loss/theft; app-based malware risks
- **Connectivity dependence** — many features require internet access

### 4G vs 5G

| Feature | 4G LTE | 5G |
|---|---|---|
| Max download speed | ~150 Mbps | ~10 Gbps |
| Latency | ~30–50 ms | <1 ms |
| Connected devices per area | Lower capacity | Much higher density |
| Use cases | Video streaming, general mobile | Autonomous vehicles, IoT, AR/VR |

## Key Terms

| Term | Definition |
|---|---|
| **Desktop computer** | Non-portable personal computer with separate components |
| **Laptop** | Portable all-in-one personal computer with built-in battery |
| **Tablet** | Touchscreen portable device, usually no physical keyboard |
| **Server** | Computer that provides services to networked client computers |
| **Mainframe** | Large, multi-user computer handling massive transaction volumes |
| **Supercomputer** | Fastest computers used for scientific simulations |
| **Embedded system** | Computer built into a device for a single dedicated purpose |
| **RTOS** | Real-Time Operating System — responds to inputs with no delay |
| **FLOPS** | Floating Point Operations Per Second — measure of supercomputer speed |
| **4G/5G** | Mobile network standards; 5G is faster with lower latency |
| **Wearable** | Computer worn on the body (watch, fitness tracker) |

## Exam Focus

> [!IMPORTANT] High-Frequency Questions
>
> 1. **"Name the type of computer and give a reason for your answer"** — identify desktop/laptop/tablet/server/embedded from a scenario
>
> 2. **"Give TWO advantages of a laptop over a desktop"** — portability, battery, all-in-one; avoid vague answers like "better"
>
> 3. **"What is an embedded system? Give an example."** — built into a device for a dedicated single purpose; example must be specific (not just "a computer in a car" but "the engine management system in a car")
>
> 4. **"Compare 4G and 5G in terms of speed and latency"** — know the numbers and what lower latency enables
