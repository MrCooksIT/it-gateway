# What is Computing?

Your phone, your laptop, the traffic light at the corner — they all process information. Before you can understand *how* they do it, you need to understand *what* a computer actually is.

---

## Definitions

| Term | Definition |
|---|---|
| **IT (Information Technology)** | The use of computers and software to manage, store, process and communicate information |
| **ICT (Information & Communication Technology)** | IT *plus* communication technology — combining computing with networks and the internet |
| **Computer** | An electronic device that accepts data as input, processes it, stores results and produces output |
| **Data** | Raw, unprocessed facts and figures (e.g. the number `37`) |
| **Information** | Data that has been processed and given meaning (e.g. "the temperature is 37°C") |
| **Knowledge** | Applying information to make decisions (e.g. "37°C is a fever — see a doctor") |

> [!NOTE] Grade 10 — Foundation
> This entire page is core Grade 10 content. Know all definitions and the computer model cold.

---

## The General Computer Model

Every computer — from a microcontroller to a high-end workstation — follows the same basic model:

| Component | Role | Examples |
|---|---|---|
| **Input** | Gets data into the computer | Keyboard, mouse, microphone, scanner, webcam, touchscreen |
| **Process** | Manipulates and transforms data | CPU (Central Processing Unit) |
| **Output** | Presents results to the user | Monitor, printer, speaker, projector |
| **Storage** | Keeps data short-term or long-term | RAM (short-term), HDD/SSD/USB (long-term) |
| **Communication** | Sends/receives data to/from other devices | Network card, modem, Wi-Fi adapter |

> [!TIP] Exam Tip
> The acronym **IPOSC** (Input → Process → Output → Storage → Communication) helps you remember the model. Most exam questions give you a device and ask which component role it plays.

## Hardware vs Software

| | Hardware | Software |
|---|---|---|
| **What it is** | Physical components you can touch | Programs and instructions the computer runs |
| **Examples** | CPU, RAM, keyboard, monitor | Windows, Delphi, MS Word |
| **Types** | Input, output, storage, processing, communication | System software, application software |
| **Can be changed?** | Must be physically replaced | Installed, updated, deleted |

---

## Data vs Information vs Knowledge

| Level | What it is | Example |
|---|---|---|
| **Data** | Raw, unprocessed facts | `37`, `John`, `Cape Town` |
| **Information** | Data processed and given context | John's temperature is 37°C |
| **Knowledge** | Using information to make decisions | 37°C is a fever — John needs to rest and see a doctor |

## Advantages of Using Computers

1. Provides access to more information
2. Completes tasks that might be impossible for humans (e.g. weather modelling, space calculations)
3. Saves time
4. Automates repetitive tasks
5. Allows for greater productivity
6. Allows for better communication and connections
7. Entertainment

---

## Disadvantages of Using Computers

| Disadvantage | Explanation |
|---|---|
| **Social risks** | Social media can be addictive, cause jealousy, and interfere with real friendships |
| **Health risks** | Excessive use causes back pain, eyestrain, obesity, **RSI**, and **CTS** |
| **Security risks** | Malware can attack systems, destroy files, steal data or allow unauthorised access |
| **High cost** | Even affordable computers are expensive for many South Africans |
| **Distractions** | High entertainment value makes it easy to lose focus and productivity |
| **Environmental impact** | Computers consume large amounts of electricity; e-waste is a growing problem |

> [!NOTE] Key Terms
> **RSI (Repetitive Strain Injury)** — painful inflammation of tendons caused by overuse (e.g. excessive typing or mouse use)  
> **CTS (Carpal Tunnel Syndrome)** — compression of the nerve in the wrist from repetitive hand/wrist movements  
> **Ergonomics** — the study of how humans interact with manufactured objects, aiming to create environments suited to users' physical needs
