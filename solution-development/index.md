# Solution Development

Solution development is the process of designing and building software to solve problems. This is the practical side of IT — covered in Paper 1 — and includes algorithm development, programming, file handling, databases in code, and OOP.

## What you will learn

| Grade | Focus |
|---|---|
| 10 | IDE basics, variables, data types, input/output, conditions, loops, basic algorithms |
| 11 | Methods, procedures, text file handling, arrays, SQL in code, event-driven programming |
| 12 | OOP (classes, objects, inheritance), relational databases in code, advanced algorithms |

## Topics in this section

- [SQL Basics](./sql-basics) — SQL SELECT, INSERT, UPDATE, DELETE in a programming context (Gr 11–12)
- [Database Theory in SQL](./database-theory-in-sql) — How database design concepts translate into code (Gr 12)

## CAPS alignment

Solution Development is the focus of Paper 1 (practical exam). It also appears in Paper 2 for algorithm questions, data structures, and general programming concepts. The PAT (Practical Assessment Task) is also based on this topic.
