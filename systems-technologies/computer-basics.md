# Computer Basics

{: .note }
This topic forms part of the Grade 10 CAPS curriculum and provides essential foundational knowledge for all IT studies.

## What are Information and Communication Technologies (ICTs)?

Information and Communication Technologies (ICTs) refer to all the technologies that enable the handling of information and facilitate different forms of communication. These include:

- Hardware devices (computers, smartphones, tablets)
- Software applications
- Network infrastructure
- Storage systems
- Communication systems (telephone networks, internet)

## Defining Information Technology (IT)

**Information Technology (IT)** is a subset of ICT that focuses specifically on:
- The use of computers and software to process, store, protect, and transfer information
- Development and management of computing systems
- Creating software applications and systems

## The General Model of a Computer

At its core, a computer is an electronic device that processes data according to a set of instructions. The general model includes:

![General Computer Model](../../../assets/images/computer-model.png)

1. **Input**: Data and instructions enter the system through input devices
2. **Processing**: The CPU (Central Processing Unit) processes the data according to instructions
3. **Storage**: Data and instructions are stored temporarily in memory or permanently in storage devices
4. **Output**: Processed information is presented through output devices
5. **Communication**: Exchange of data between systems through communication devices

## Components of a Computer System

### Hardware Components

Hardware refers to the physical components of a computer system:

| Component Category | Examples | Function |
|-------------------|----------|----------|
| Input Devices | Keyboard, mouse, scanner, camera | Allow data entry into the system |
| Processing Units | CPU, GPU | Execute instructions and process data |
| Storage Devices | Hard drives, SSDs, USB drives | Store data permanently |
| Memory | RAM, cache | Store data temporarily during processing |
| Output Devices | Monitor, printer, speakers | Present processed information |
| Communication Devices | Network card, modem, router | Enable data exchange between systems |

### Software Components

Software refers to the programs and data that give instructions to the hardware:

1. **System Software**
   - Operating Systems (Windows, macOS, Linux)
   - Utility Programs (disk management, antivirus)
   - Device Drivers

2. **Application Software**
   - Productivity software (word processors, spreadsheets)
   - Graphics software
   - Entertainment software
   - Custom applications

## Types of Computers

Computers can be categorized based on size, power, and purpose:

### By Size and Portability

- **Supercomputers**: Extremely powerful machines used for complex calculations
- **Mainframes**: Large computers used by organizations for critical applications
- **Desktop computers**: Personal computers designed for use at a single location
- **Laptops/Notebooks**: Portable personal computers
- **Tablets**: Portable touchscreen devices
- **Smartphones**: Handheld devices with computing capabilities
- **Embedded computers**: Special-purpose computers built into other devices

### By Purpose and Usage

- **Servers**: Provide services to other computers on a network
- **Workstations**: Powerful desktop computers for specialized work
- **Personal computers**: General-purpose computers for individual use
- **Thin clients**: Computers that rely on servers for processing
- **Gaming computers**: Optimized for playing video games
- **Industrial computers**: Designed for manufacturing and industrial environments

## Advantages and Disadvantages of Using Computers

### Advantages

1. **Speed**: Computers can process vast amounts of data in seconds
2. **Accuracy**: When programmed correctly, computers make fewer errors than humans
3. **Storage Capacity**: Modern computers can store enormous amounts of data
4. **Automation**: Repetitive tasks can be automated
5. **Connectivity**: Computers enable global communication and information sharing

### Disadvantages

1. **Cost**: Initial setup and maintenance can be expensive
2. **Security Concerns**: Vulnerability to cyberattacks and data breaches
3. **Dependence**: Over-reliance on technology can be problematic if systems fail
4. **Environmental Impact**: Manufacturing and disposal of computer equipment can harm the environment
5. **Health Issues**: Prolonged computer use can lead to physical and mental health problems

## The ICT System Concept

An **ICT system** is broader than just a computer. It includes:

1. The hardware and software components
2. The people who use, maintain, and develop the system
3. The procedures for using the system
4. The data processed by the system
5. The communication networks connecting components

A typical ICT system follows this process:
- **Convey**: Transport data from one place to another
- **Manipulate**: Process the data according to specific rules
- **Store**: Retain the data for future use
