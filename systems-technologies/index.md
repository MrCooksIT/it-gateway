# Systems Technologies

Systems technologies cover the physical and non-physical components of a computer system — hardware, peripherals, and software — and how they work together to perform computing tasks.

## What you will learn

| Grade | Focus |
|---|---|
| 10 | Basic hardware components, types of computers, operating systems, utility software |
| 11 | Motherboard detail, storage, memory, OS management, security threats |
| 12 | Mobile technologies, performance factors, cloud computing, virtualisation, AI/AR/VR |

## Topics in this section

- [Computer Basics](./computer-basics) — ICT concepts, the general computer model, types of computers
- [Data Representation](./data-representation) — Binary, hexadecimal, ASCII, Unicode
- [Mobile Technologies](./mobile-technologies) — Mobile devices, constraints, performance factors
- [Computer Security](./computer-security) — Security threats, cybercrime, protection methods
- [Cloud Computing](./cloud-computing) — Cloud services, virtualisation, emerging technologies

## CAPS alignment

This topic appears in the theory paper (Paper 2) across all three grades. It covers both hardware and software concepts and connects directly to the Solution Development topic through operating systems, IDEs, and development environments.
